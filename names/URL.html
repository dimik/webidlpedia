Definition

URL Standard defines URL

[Exposed=(<a href='Window.html'>Window</a>,<a href='Worker.html'>Worker</a>),
 LegacyWindowAlias=<span>webkitURL</span>]
interface <strong>URL</strong> {
  constructor(<span>USVString</span> <strong>url</strong>, optional <span>USVString</span> <strong>base</strong>);
  stringifier attribute <span>USVString</span> <strong>href</strong>;
  readonly attribute <span>USVString</span> <strong>origin</strong>;
           attribute <span>USVString</span> <strong>protocol</strong>;
           attribute <span>USVString</span> <strong>username</strong>;
           attribute <span>USVString</span> <strong>password</strong>;
           attribute <span>USVString</span> <strong>host</strong>;
           attribute <span>USVString</span> <strong>hostname</strong>;
           attribute <span>USVString</span> <strong>port</strong>;
           attribute <span>USVString</span> <strong>pathname</strong>;
           attribute <span>USVString</span> <strong>search</strong>;
  [SameObject] readonly attribute <a href='URLSearchParams.html'>URLSearchParams</a> <strong>searchParams</strong>;
           attribute <span>USVString</span> <strong>hash</strong>;

  <span>USVString</span> <strong>toJSON</strong>();
};

This interface is extended in the following specifications:

  1. File API 
    [Exposed=(<a href='Window.html'>Window</a>,<span>DedicatedWorker</span>,<a href='SharedWorker.html'>SharedWorker</a>)]
partial interface <strong>URL</strong> {
  static <span>DOMString</span> <strong>createObjectURL</strong>((<a href='Blob.html'>Blob</a> or <a href='MediaSource.html'>MediaSource</a>) <strong>obj</strong>);
  static <span>undefined</span> <strong>revokeObjectURL</strong>(<span>DOMString</span> <strong>url</strong>);
};
  2. Media Source Extensions™ 
    [Exposed=<a href='Window.html'>Window</a>]
partial interface <strong>URL</strong> {
    static <span>DOMString</span> <strong>createObjectURL</strong> (<a href='MediaSource.html'>MediaSource</a> <strong>mediaSource</strong>);
};
