Definition

WebGL Specification defines WebGLContextAttributes

dictionary WebGLContextAttributes {
  boolean alpha;
  boolean depth;
  boolean stencil;
  boolean antialias;
  boolean premultipliedAlpha;
  boolean preserveDrawingBuffer;
  WebGLPowerPreference powerPreference;
  boolean failIfMajorPerformanceCaveat;
  boolean desynchronized;
};

This dictionary is extended in the following specifications:

  1. WebXR Device API 
    partial dictionary WebGLContextAttributes {
  boolean xrCompatible;
};
